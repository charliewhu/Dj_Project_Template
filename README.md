# Django Project Template

Readymade Django project, complete with:
* Poetry ✅
* Gitignore ✅
* Github Actions ✅
* Custom User model ✅
* Folder restructure ✅
* Static setup ✅
* Environs[django] ✅
* Behave-django for BDD ✅
* Model-Bakery ✅
* Psycopg2 ✅
* Gunicorn ✅
* Dockerfile ✅
* docker-compose ✅
* Nginx ✅

## Creating a project

```bash
// clone repo
mkdir your_folder_name
cd your_folder_name
git clone https://github.com/charliewhu/Dj_Project_Template.git .

// run development server
docker-compose up
```
